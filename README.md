# openspec-demo

Hands-on introduction to cursor and openspec

## Première utilisation (après `git clone`)

**macOS** : Terminal. **Windows** : PowerShell ou Invite de commandes. Les étapes sont les mêmes.

1. **Node.js** 20.19+ — [nodejs.org](https://nodejs.org/) (installateur LTS pour Mac ou Windows).
2. `npm install -g @fission-ai/openspec@latest`
3. _(optionnel)_ `openspec completion install`
4. Ouvrir ce dépôt dans **Cursor**.

Après mise à jour du CLI : `openspec update` à la racine. Pas besoin de `openspec init` ici (déjà versionné) ; exception : `openspec init --tools cursor` pour tout régénérer.

Détail : [docs/commandes-openspec-init.md](docs/commandes-openspec-init.md).
